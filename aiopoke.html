An asynchronous API wrapper for the PokeApi. It is designed to be easy to use and and intuitive. An example:

aiopoke.py

import asyncio
import aiopoke

async def main() -> None:
    async with aiopoke.AiopokeClient() as client:
        berry = await client.get_berry(1)
        print(berry)

        # An object can also have 'MinimalResources' as attributes
        # You can fetch those with .fetch(), for example:
        item = await berry.item.fetch()

asyncio.run(main())
